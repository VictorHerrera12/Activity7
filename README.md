# Activity 7 - Robotics Platform

## Descripción
Este proyecto consiste en el desarrollo de una plataforma para la gestión de cursos de robótica. Permite la administración de usuarios, grupos y cursos, así como la asignación de cursos a grupos.

## Tecnologías
- Laravel
- PHP
- MySQL
